---

## ฟีเจอร์

### 1. แนะนำตัว

แสดงข้อมูลส่วนตัว ประวัติการทำงาน และเหตุผลที่ต้องการเข้าร่วม DevPool

### 2. Dark Mode

ผู้ใช้สามารถกดปุ่มเพื่อสลับระหว่างโหมดสว่างและโหมดมืดได้

### 3. แสดงข้อมูลเพิ่มเติม

กดปุ่มเพื่อแสดงหรือซ่อนข้อมูลเพิ่มเติมเกี่ยวกับความสนใจและการเรียนรู้ด้านเทคโนโลยี

### 4. Live Input

เมื่อผู้ใช้พิมพ์ข้อความ ระบบจะแสดงข้อความที่พิมพ์แบบทันทีบนหน้าเว็บ

### 5. Responsive Design

รองรับการแสดงผลทั้งบนคอมพิวเตอร์และอุปกรณ์มือถือ
